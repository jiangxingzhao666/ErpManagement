## 关键测试场景

| 场景 | 操作 | 预期结果 |
|------|------|----------|
| 库存预警 | 打开 StockAlert.aspx | 显示9条预警商品 |
| 软删除商品 | admin 打开商品列表 | 显示28条（含3条灰色） |
| 软删除商品 | manager 打开商品列表 | 只显示25条正常商品 |
| 已禁用用户 | admin 打开用户管理 | 显示7条（含2条禁用） |
| 已禁用用户 | manager 打开用户管理 | 只显示5条正常用户 |
