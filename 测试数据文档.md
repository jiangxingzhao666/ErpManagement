## 关键测试场景

| 场景 | 操作 | 预期结果 |
|------|------|----------|
| 前台购物 | 访问 http://localhost/ → 添加商品 → 结算 | 结算成功 + 快递选择按钮 |
| 商品图片 | 浏览商品列表（前台/后台） | 所有 25 个商品显示对应 SVG 图片 |
| 图片上传 | 编辑商品 → 选择文件 → 保存 | 图片保存到 Content/images/products/ |
| 操作员权限 | staff 登录 | 自动跳转购物车，侧边栏仅购物车入口 |
| 店长权限 | manager 登录 → 用户管理 | 只能看到操作员角色选项 |
| 管理员权限 | admin 登录 → 全部页面 | 可见软删除商品和已禁用用户 |
| 快递创建 | 结算后点击"创建快递单" | 已登录→Edit.aspx, 未登录→Public.aspx |
| 快递状态 | 快递列表 → 点击发货/签收 | 状态: 待发货→运输中→已签收 |
| 会员升级 | 客户累计消费达 5000/10000 | 自动升级为银卡/金卡 |
| 库存变动 | 销售/进货操作后查看 | 库存数量正确增减 |
