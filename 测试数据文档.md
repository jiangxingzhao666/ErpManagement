## 关键测试场景

| 场景 | 操作 | 预期结果 |
|------|------|----------|
| 前台页面 | 访问 http://localhost/ | 显示超市介绍 + 商品卡片 + 购物车 |
| 商品图片 | 前台/后台浏览商品 | 显示对应 SVG 图片 |
| 操作员权限 | staff 登录 | 自动跳转购物车，侧边栏仅购物车 |
| 店长权限 | manager 登录 | 全部页面可见，用户管理仅可操作"操作员" |
| 管理员权限 | admin 登录 | 全部页面 + 可见软删除数据 |
| 商品编辑 | 管理员编辑商品 | 可上传/删除商品图片 |
| 库存预警 | 查看 Stock 页面 | 显示库存 ≤ 最低库存的商品 |
